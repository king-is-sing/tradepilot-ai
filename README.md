# TradePilot AI

TradePilot AI is a crypto market scanner and paper-trading agent using Kraken public market data.

It scans selected crypto pairs, calculates price movement, volume spikes, spread, and simple momentum signals. It starts in paper-trading mode only.

## Important

This project does not provide financial advice.
This project does not guarantee profit.
Crypto is high risk.
The first version does not place real trades.

## Current features

- Kraken public market data
- Momentum scanner
- Volume spike detection
- Spread filter
- Paper trading simulation
- Basic risk controls

## Planned features

- AI risk filter
- Dashboard
- Backtesting
- Alerts
- Manual trade approval
